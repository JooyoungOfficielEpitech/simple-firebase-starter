# Musical Theatre App - UX Documentation Index

## Purpose of This Documentation

This documentation set provides a comprehensive UX analysis of a musical theatre application to support a complete redesign effort. Each document examines specific user flows, interactions, and design considerations to help identify opportunities for improvement and ensure a cohesive user experience.

---

## App Overview

### Core Features

The app serves the musical theatre community with two primary features:

1. **Bulletin Board System**
   - Musical gathering announcements
   - Theatre company recruitment posts
   - Audition and performance notifications

2. **Music Player & Metronome**
   - Practice tool for actors
   - Audio playback capabilities
   - Integrated metronome functionality

### User Types

**Actors**
- View gathering announcements and recruitment posts
- Practice musical pieces using the music player
- Use metronome for rhythm practice

**Theatre Companies & Organizations**
- Post recruitment announcements
- Advertise musical gatherings
- Connect with potential cast members
