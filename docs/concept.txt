🎭 뮤지컬 노래방 (가칭) - PRD v0.1 (Beta 버전 목표)
1. 서비스 개요

목적: 뮤지컬 넘버를 쉽게 부르고 연습할 수 있도록, 노래방과 유사한 UX를 제공하는 모바일 앱.

핵심 가치:

MR + 가사 싱크를 통한 몰입감 있는 연습

실시간 음정 분석으로 정확한 가창 가이드 제공

뮤지컬 연습을 "게임처럼 재미있게"

2. 타겟 사용자

1차 타겟: 뮤지컬 동호회, 아마추어 뮤지컬 배우, 뮤지컬 전공 학생

2차 타겟: 뮤지컬 팬덤(관객 → 직접 불러보고 싶은 사용자)

3. MVP 핵심 기능

곡 등록 (관리자 전용)

MR(mp3/wav), 악보(MusicXML) 업로드 → Firebase Storage 저장

곡 메타데이터 (제목, 가사 싱크, 음정 데이터) → Firestore 저장

노래 실행 (사용자)

MR 재생 + 가사 싱크 (노래방 UI)

가사 진행 시 현재 파트 하이라이트 표시

실시간 음정 분석

마이크 입력 → 음정 분석 (FFT + pitch detection)

원곡 음정과 비교 → "낮음/높음/정확" 시각화

4. 추후 확장 기능 (MVP 이후)

노래 종료 후 점수/피드백 제공

사용자 기록 저장 (부른 곡, 점수 히스토리)

커뮤니티/랭킹

구독 모델 도입 (프리미엄 곡 무제한 이용)


5. 사용자 시나리오 (Beta 기준)

관리자가 MR + MusicXML 업로드 → Firebase 저장

사용자가 앱에서 곡 선택

MR이 재생되며 가사 싱크 UI 표시

사용자가 노래를 부르면 → 마이크 입력으로 음정 분석 → 화면에 "높음/낮음/정확" 피드백

노래가 끝나면 → 단순히 종료 (Beta 버전에서는 점수 없음)
